MineMod VulnIndexer

MineModVulnIndexer was a passive Python prototype for indexing public Minecraft mod metadata, recent public fix signals, and exact modpack component versions. The repository is archived and kept for reference only.

What It Did

  • Retrieve public CurseForge and Modrinth metadata.
  • Identify official source repositories.
  • Collect public changelog, issue, PR and commit signals.
  • Correlate previous mod releases with indexed modpack components.
  • Display local SQLite results in a read-only Streamlit dashboard.

Safety Scope

This project does not scan Minecraft servers, connect to multiplayer servers, execute proof-of-concepts or exploit third-party systems. It does not mirror or redistribute mod files.

Status

Archived. The final prototype could crawl and triage public metadata, but signal quality was inconsistent and no production-ready workflow was reached.
