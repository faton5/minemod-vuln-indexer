# Local Lab Example

Use this folder for notes, manifests or disposable configuration related to a
local Minecraft validation server that you control.

Valid targets are local or private addresses such as:

- `127.0.0.1:25565`
- `localhost:25565`
- `192.168.1.50:25565`

Public server discovery and exploit testing against third-party systems are out
of scope for this project.
